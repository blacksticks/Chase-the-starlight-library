# -
此数据库用于深海迷航《星海逐光》整合包项目的相关自创模组

《星海逐光》是一个基于《深海迷航Subnautica》的游戏整合包，整合包在原版游戏的基础上，添加了大量的新剧情、新科技、新生物群系......以及更多！

此数据库用于自创模组的项目管理。目前项目进度如下：

Just Enough Items ———— 深海迷航JEI配方管理工具模组（已完成）
  ●此模组替换了游戏PDA中的配方页面，支持玩家点击物品，查看物品的相关来源与用途（已实现）
  ●添加物品检索，方便快速查找需要的物品（已实现）
  ●蓝图追踪功能接入（未完成）
  ●UI重置（未完成）


Quest Book ————  深海迷航任务书（开发中）
  ●此模组提供了一个可交互的任务书界面，开发者可以自己制定章节与任务链，为整合包玩家提供一个明确的游戏进度指示。玩家可以按照开发者提供的任务进度推进游戏，完成任务，领取奖励
  ●主体UI开发（基本完成）
  ●章节页签及相关功能开发（基本完成）
  ●任务链页签以及相关功能开发（开发中）
  ●任务与奖励条件（部分完成）
  ●其他辅助功能与设置（数据持久化已完成）


Research System ————  科研系统（计划）
  ●此模组替代原本的碎片扫描制度，通过建造科研设备，提供科研点数，利用科研点数解锁科技链从而推进科技与蓝图解锁
  ●各游戏阶段科研设备设计（已完成）
  ●各游戏阶段科研设备数值（未完成）
  ●科研设备运行代码段（未完成）
  ●科研设备UI及建模（未完成）


Chase The Starlight Lib  ————  星海逐光整合包数据库
  ●此数据库用于存放整合包的全部相关物品、设备、工具、载具的图标、建模以及相关功能代码段
  ●全部物品设计（已完成）
  ●全部物品建模及图标（未完成）
  ●部分物品特殊功能（未完成）


Chase The Starlight assist  ————星海逐光整合包辅助模组
  ●此模组用于游戏演出、相关辅助功能与结局修改相关内容
  ●游戏演出（未完成）
  ●生物群系提醒（未完成）
  ●结局修改（未完成）
